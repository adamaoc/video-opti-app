# VidOpti

Local desktop app for compressing and resizing phone videos. Drop files, pick a preset, get smaller MP4s — no cloud, no editing.

## Features

- Drag-and-drop or file picker for MP4, MOV, M4V, WEBM, and more
- Presets: Quick share (720p), Social (1080p), Smallest (480p), Custom
- Batch queue with per-file progress
- Output beside source file or to a chosen folder
- macOS packaging ready

## Requirements

- Node.js 20+
- macOS (primary target for v1)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
